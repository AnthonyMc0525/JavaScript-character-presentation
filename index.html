Creating Basic Characters in JavaScript

By: Anthony McMellen

The characters are gonna be used in turn based combat systems like Final Fantasy or Pokemon

Each character will be a JavaScript "Class"

Classes are a collection of information about what a thing is and what it can do

So for our character, the Class will hold these:
  • HP
  • ATK
  • DEF
  • MP
It will also hold things like spells or attacks and how they will behave.
To get started, we put the Class into the story javascript file 
                class BlackMage{

                }
Next we give the character the stats that we mentioned before
For this we are gonna use "constructor()" which says what you need to input.
in this example, I made numbers for each stat to make it easier 
              class BlackMage{
                constructor(HP=1243, MP=1647, DEF=60, ATK=100){
                  this.HP = HP;
                  this.MP = MP;
                  this.DEF = DEF;
                  this.ATK = ATK;
                }
              }
Now that we have the stats for the character we can add an attack.
I'll add the spell "Firaga" to this class. it is going to be a set of instructions called a function.

The function should look like this:

                firaga(target){
                  target.HP -= (some amount of health)
                  this.MP -=24;
                }

(target) is the person you are trying to hit. in this function, you are doing damage to the target and you are using mp to do it.

Now that we have that lets see what it should look like when you put it all together:
            class BlackMage{
              constructor(HP=1243, MP=1647, DEF=60, SPR=100){
                this.HP = HP;
                this.MP = MP;
                this.DEF = DEF;
                this.SPR = SPR;

                firaga(target){
                  target.HP -= (some amount of health)
                  this.MP -=24;
                }
              }
            }
And there you have it. A simple character for a turn based battle system
